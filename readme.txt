Tiled JSON to Cheese Engine Map File Converter V 1.0
By Geibu3D
----------------------------------------------------


This converter converts .json files exported from the program Tiled and converts them to a .csv/line data format that the Cheese Engine can read.

To Use:

Drag and drop a .json file on top of the executable and it will output the converted file. You may also use the command line.

Note: This tool is completely useless to anyone but I, who possesses the mighty Cheese Engine.
